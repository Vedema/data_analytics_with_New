Project 2: Retail Customer SQL Analysis

Relational Database Design & Business Intelligence Using SQL

📌 Project Overview

This project demonstrates the design and implementation of a relational retail database using SQLite. The database was structured using primary and foreign key relationships to eliminate redundancy and ensure data integrity. SQL queries were written to analyse customer behaviour, product performance, and revenue trends.

📈 Business Insights

  • London customers contributed the largest revenue.
  • Laptops generated more sales (£900) than phones (£600).
  • July showed consistent revenue across weeks.
  • High-value products significantly influence overall revenue.
